# octree-visualiser
Visualiser for a 3D spatial partitioning acceleration data structure; the octree.

## Getting Started
This section will guide you through the process of setting up **octree-visualiser**.

### Requirements
This project is developed on Linux (Arch Linux) and requires the following system packages:
- [glfw 1:3.4-1](https://archlinux.org/packages/extra/x86_64/glfw/)

### Installation

Run the project:
```sh
./scripts/install.sh && ./scripts/run.sh
```
